# Gercotronic Website

A modern, multi-page corporate website built with HTML, TailwindCSS, and modular design principles.

## Features

- Multi-page architecture (Services, Packages, Projects, About, Contact)
- Responsive design with TailwindCSS
- Clean navigation structure optimized for GitHub Pages deployment
- Legal compliance pages (Impressum & Datenschutz)
- Scalable component-based layout

## Tech Stack

- HTML5
- TailwindCSS (CDN)
- Vanilla JavaScript
- GitHub Pages (deployment)
